openxava / documentation / Example applications with source code

OpenXava is a powerful and versatile tool for agile development of enterprise applications. While many applications developed with OpenXava are commercial and their code is not publicly available, fortunately, we have a series of open-source examples that offer a transparent view of how OpenXava works in practice. Here are some of them.
Also look at Project Templates as a simple way to get applications with source code.

Invoicedemo y openxavatest

Invoicedemo is a simple example of a billing application developed with OpenXava. This project demonstrates how basic invoicing functionalities can be implemented using the platform.
Openxavatest is a more complex example that covers a wide variety of use cases and functionalities. Used as a testing tool for OpenXava, this project provides concrete examples of how the platform can handle more complex and demanding situations in enterprise application development.
You can visit the repository.

You can also watch the video on how to clone the repository and import the openxavatest project.

Invoicing (OpenXava course)

The OpenXava course repository offers a developed application. This is invaluable for those who want to learn the basics of OpenXava and how to apply them in real projects. From entity creation to the implementation of advanced functions, this sample application provides a comprehensive view of the technology's potential.

Xavaprojects

Xavaprojects is an example of a project management application developed with OpenXava. This project illustrates how the platform can be used to create practical and functional solutions for business environments, facilitating collaboration and coordination in projects of any scale.

Xavacrm

Xavacrm is a basic example of a small customer relationship management (CRM) system developed with OpenXava.

ZX-Ed

ZX-Ed is a database editor and script generator for ZXDB.
