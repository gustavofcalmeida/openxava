openxava / documentación / Aplicaciones de ejemplo con código fuente

OpenXava es una herramienta potente y versátil para el desarrollo ágil de aplicaciones empresariales. Aunque muchas de las aplicaciones desarrolladas con OpenXava son comerciales y su código no está disponible públicamente. Por suerte, contamos con una serie de ejemplos de código abierto que ofrecen una visión transparente de cómo funciona OpenXava en la práctica. Aquí están algunos de ellos.
También mira las Plantillas de Proyecto como una forma simple de obtener aplicaciones con el código fuente.

Invoicedemo y openxavatest

Invoicedemo es un ejemplo simple de una aplicación de facturación desarrollada con OpenXava. Este proyecto muestra cómo se pueden implementar las funcionalidades básicas de facturación utilizando la plataforma.

Openxavatest es un ejemplo más complejo que abarca una amplia variedad de casos de uso y funcionalidades. Utilizado como herramienta de prueba para OpenXava, este proyecto proporciona ejemplos concretos de cómo la plataforma puede manejar situaciones más complejas y exigentes en el desarrollo de aplicaciones empresariales.

Facturación (curso de OpenXava)

El repositorio del curso de OpenXava ofrece una aplicación desarrollada. Este es invaluable para aquellos que desean aprender los conceptos básicos de OpenXava y cómo aplicarlos en proyectos reales. Desde la creación de entidades hasta la implementación de funciones avanzadas, este aplicación de ejemplo proporciona una visión completa del potencial que tiene OpenXava.

Xavaprojects

Xavaprojects es un ejemplo de una aplicación de gestión de proyectos desarrollada con OpenXava. Este proyecto ilustra cómo la plataforma puede utilizarse para crear soluciones prácticas y funcionales para entornos empresariales, facilitando la colaboración y la coordinación en proyectos de cualquier escala.

Xavacrm

Xavacrm es un ejemplo básico de un pequeño sistema de gestión de relaciones con los clientes (CRM) desarrollado con OpenXava.

ZX-Ed

ZX-Ed es un editor de bases de datos y generador de scripts para ZXDB.
